# InnoDBOptimizationTree
Mysql InnoDB存储引擎性能调优


<pre>
InnoDB Buffer Pool

          用户对数据库的最基本要求就是能高效的读取和存储数据，但是读写数据都涉及到与低速的设备交互，为了弥补两者之间的速度差异，
      所以有数据库都有缓存吃，用来管理相应的数据页，提高数据库的效率，当然也因为引入了这一中间层，数据库对内存的管理变得相对复杂。
</pre>
